=========================
Hospital Management Visit
=========================

Odoo module for recording patient visits and managing hospital daily operations.

Features
========

* Patient and Doctor management.
* Scheduling and rescheduling visits.
* ICD-10 Disease registry.
* Medical diagnoses tracking.
* Doctor schedule generation.
* Access rights for different user roles (Patient, Intern, Doctor, Manager, Administrator).
